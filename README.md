# Fencing Assistant

A mobile-first fencing contractor toolkit built with React, TypeScript, Vite, Tailwind CSS, React Konva, Zustand, Zod, and React Hook Form.

## Current tool

- Gate Calculator
- Interactive structural drawing
- Featheredge board layout
- Frame parts and cut list
- 4.8 m stock optimisation
- PNG and PDF export

## Development

```bash
pnpm install
pnpm dev
```

## Production build

```bash
pnpm build
```
